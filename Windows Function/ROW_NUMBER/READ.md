# Casos de utilização
É utilizado para comparação entre linhas/janelas, sem repetição do número de rank quando encontrar valores repetidos, com número sempre sequencial de 1 a N.
Ex:
|id|valor unitário|RANK|
|--|--------------|----|
|1 |500           |1   |
|1 |500           |2   |
|1 |300           |3   |
|1 |150           |4   |
|1 |100           |5   |
